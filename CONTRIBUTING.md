# Katkıda Bulunma Rehberi

Bu projeye katkıda bulunmak istediğiniz için teşekkürler! Logo ERP Geliştirici Rehberi, topluluk desteğiyle büyüyen bir projedir.

## Nasıl Katkıda Bulunabilirsiniz?

1. **Hata Bildirimi:** Dokümanlarda gördüğünüz hataları veya eksiklikleri "Issues" sekmesinden bildirebilirsiniz.
2. **Yeni İçerik:** Eksik olan tabloları veya yeni SQL örneklerini ekleyebilirsiniz.
3. **Düzeltmeler:** Yazım hataları veya format bozukluklarını düzeltebilirsiniz.

## Pull Request Süreci

1. Bu repoyu fork'layın.
2. Kendi fork'unuzda yeni bir branch oluşturun (`git checkout -b ozellik/yeni-tablo-ekleme`).
3. Değişikliklerinizi yapın ve commit'leyin.
4. Branch'inizi push'layın (`git push origin ozellik/yeni-tablo-ekleme`).
5. GitHub üzerinden bir Pull Request (PR) oluşturun.

## Kod/Doküman Standartları

- Markdown dosyalarında standart GitHub formatını kullanın.
- SQL sorgularında açıklama satırları (`-- Açıklama`) eklemeyi unutmayın.
- Tablo isimlerini büyük harfle yazın (`LG_001_ITEMS`).

Teşekkürler!
