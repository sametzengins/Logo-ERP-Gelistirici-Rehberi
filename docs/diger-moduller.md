# Diğer Modüller

Banka, Kasa, Hizmet, Sabit Kıymet ve diğer yardımcı modüller.

## Banka Yönetimi
| Tablo Adı | Açıklama |
|-----------|----------|
| `LG_XXX_BNCARD` | Banka Kartları |
| `LG_XXX_BANKACC` | Banka Hesapları |
| `LG_XXX_XX_BNFICHE` | Banka Fişleri |
| `LG_XXX_XX_BNFLINE` | Banka Hareketleri |
| `LG_XXX_XX_BNTOTFIL` | Banka Aylık Toplamları |

## Kasa Yönetimi
